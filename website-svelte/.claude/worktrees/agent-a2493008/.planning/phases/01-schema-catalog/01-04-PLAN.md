--- LazyImage.svelte ---
Implements intersection-observer-gated lazy loading with placeholder and fade-in.
Props: `src: string | null`, `alt: string`.

- Import `useIntersectionObserver` from `'runed'`
- Import `ImageOff` from `'lucide-svelte'`
- Use a bound `imgEl` element ref ($state<HTMLElement | undefined>)
- `isVisible` ($state boolean) flips to true on first intersection entry
- `loaded` ($state boolean) flips to true on img `onload`
- `errored` ($state boolean) flips to true on img `onerror`
- Outer div: `class="relative h-40 w-full overflow-hidden bg-stone-100"`
- Placeholder div (shown when !src || errored || !isVisible): `class="absolute inset-0 flex items-center justify-center text-stone-300"`; contains `<ImageOff size={32} />`
- Image (shown when src && isVisible && !errored): `loading="lazy"`, `class="h-full w-full object-cover transition-opacity duration-200 {loaded ? 'opacity-100' : 'opacity-0'}"`
- Bind the `imgEl` ref to the image element so useIntersectionObserver can observe it
